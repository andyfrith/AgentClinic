# Tech Stack

| Layer | Technology |
|---|---|
| Framework | Next.js (App Router) |
| Bundler / Dev Server | Vite |
| Language | TypeScript |
| Database | PostgreSQL (local via Docker) |
| ORM | Drizzle |
| Data Fetching | TanStack Query |
| Validation | Zod |
| Forms | React Hook Form |
| Component Library | shadcn/ui |
| Styling | Tailwind CSS |
| Animation | Framer Motion |
| Unit Testing | Vitest |
| E2E Testing | Playwright |
| UX | Great animation & polish throughout |

## Principles

- Full type safety from database to UI
- Server components for data fetching, client components for interactivity
- **Responsive first** — all UI adapts gracefully across mobile (320px+), tablet, and desktop viewports using Tailwind responsive prefixes. Content reflows, navigation stays usable, and touch targets remain large enough at every breakpoint.
- Accessible and performant by default
- Animations should feel natural and purposeful, never gratuitous
